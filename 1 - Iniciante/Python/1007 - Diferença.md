#Pergunta:

Leia quatro valores inteiros A, B, C e D. A seguir, calcule e mostre a diferença do produto de A e B pelo produto de C e D 
segundo a fórmula: DIFERENCA = (A * B - C * D).
```

```
#Resposta:

a = int(input())
b = int(input())
c = int(input())
d = int(input())
dif = (a * b) - (c * d)

print("DIFERENCA = {}".format(dif))
